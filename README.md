## Requirements
* Minetest 5.3.0 (or later)
* default mod

## Supported mods
* sfinv
* xdecor

## Settings
See the settings in the advanced settings menu of the Minetest GUI or in [minetest.conf.example](https://github.com/BrunoMine/craft_table/blob/master/minetest.conf.example).
